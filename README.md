## Project Overview

Notified is a web application for users to organize their tasks by level or urgency.

<img width="1128" height="733" alt="Image" src="https://github.com/user-attachments/assets/bdc3c04b-4261-4d12-a4c3-b42ed9026fd8" />


## Features

- Interactive userface for users to add items in the category of their choosing.
- Ability to clear items by category and from the entire application as you finish your tasks.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Notified.git
````

2. Navigate to the project directory:

```bash
cd Notified
```

3. Open Project File

4. Use in browser
   
## Usage

1. Add tasks to the urgency category of your choice.
2. Clear items as they are completed.
